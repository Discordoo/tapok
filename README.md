# Metadata preprocessor for Discordoo documentation

Usage
-
```shell
$ tapok path/to/markdown/files
```

Examples
-
```shell
$ tapok ./docs
# ┗ will add metadata to all .md files in subdirs classes/ interfaces/ enums/ modules/
```
